CAR2 LOOP DRIVE 仕様まとめ（2026-07-18 / v20260718-car2-48）
この会話で実装した内容の最終版。廃止済みの仕様は除外。
====================================================

■ キー操作（追加・変更分）
  T   : 全体地図（旧Mから移動）
  M   : 音楽選択モード
  N   : 夜間モード ON/OFF
  B   : 音声切替 3状態（エンジン音 → 車内音 → 無音 → …）
  V   : ボンネットカメラ（一人称視点）
  F   : バックミラー（画面上部の小窓に後方視点）
  Y   : 自動運転
  G   : 表示切替 4状態（全表示 → 操作説明削除 → 曲名削除 → スピード表示削除 → 全表示）
  J/K/L : 曲の10秒巻き戻し / 再生・一時停止 / 10秒早送り（運転中も有効）
  ESC : ゲーム一時停止（PAUSE表示＋操作説明パネルを表示）

■ CPU車
  ・道路外に出ない: 走行ラインから横±2.5mに毎フレーム位置を制限。
    ドリフトで膨らんでも壁の外へは出ない（レイキャスト復帰は保険として残存）
  ・ドリフトで曲がる: 車体が先にコーナーへ切れ込み、進行方向が遅れて追従。
    最大約30°の横滑り角。ドリフト中はロール強化＋タイヤスモーク（自車の近くのみ）
  ・テールランプ残像: CPU車のみ（自車は出さない）。
    点ではなく線（前回位置と繋ぐ帯・横グラデーション）で連続した赤い光の筋に見せる。
    0.5m毎・約0.55秒でフェード・自車から130m以内のみ・周回ワープ時は繋がない

■ 自車
  ・夜間は toyota86n.vox（ライト拡大版）へ自動切替。
    フロントの黄色部分（純黄255,255,51 と 淡黄255,255,153 の両方）が発光。
    判定はRGB比率式なので同系統の黄色を塗り足せば自動で発光対象になる
  ・ランプ発光はランプ部分の小さな光点のみ（ボンネットは光らない配置）
  ・前方約5mだけ照らす実光源スポットライト。
    光源を車体前端より前に置いて前方だけへ向けるため、自車の車体には光が当たらない
  ・テールランプ残像はなし

■ フォグ
  ・開始130m→500m / 空色に溶ける距離480m→1150m に変更。
    近〜中距離の色の霞みを解消しつつ、遠景のぼかし（周回コピーの切れ目隠し）は維持

■ 夜間モード（N）
  ・空・フォグはほぼ黒に近い濃紺（0x050a12）。環境光0.13・太陽光0.04
  ・グレー路面のみ昼の1/3程度の明るさを保つ（路面マテリアルへ自己発光。壁・建物は暗いまま）
  ・道路外の純白マテリアル（FFFFFF）は発光（窓明かり用）。
    SketchUpでビルに純白を追加して map.gltf を書き出せば自動で光る

■ 街灯
  ・車7台分（33.6m）間隔・左右同じ列・壁の少し外側（路面端から約1.5〜3.5m）。
    路面判定（レイキャスト）で自動配置し、道路上には絶対に立てない。
    路面外へ出られない場所はスキップ（現在249本）
  ・高さは車3台分（約14.4m）。直立ポールの頂点が道路側へ折れて短い水平アーム、
    先端に地表を向いた平らなランプ
  ・夜は先端が白く発光＋白ハロ（光量4倍）＋地表の光だまり（照射範囲3倍・濃さ1/2）
  ・InstancedMeshで前後の周回コピー分も含めて一括描画（当たり判定なし）

■ 路面判定（白線対応）
  ・白線（白系マテリアルの高さ0付近の面）も走行可能面として扱う
    （白線上で「路面外」と誤判定され引き戻される＝引っ掛かる問題の修正）
  ・高さ30cm以下の低い段差（白線・ペイントの側面）は壁とみなさない

■ 音声切替（B）
  ・エンジン音 → 車内音 → 無音 → …の3状態
  ・車内音: 80km/h未満 = sound/drive_on_freeway1.mp3 /
            80km/h以上 = sound/drive_on_freeway2.mp3 をループ再生（±2km/hの余裕で切替）
  ・どのモードでもドリフト音（タイヤスクリーチ）は鳴る（エンジン系のみミュート）
  ・ドリフト音の音量係数は0.6（従来の1.5倍）

■ 音楽選択モード（M）
  ・musiclist.txt の「」内ラベルとURLのみ解析（色コード等の他の文言は無視）
  ・開いている間は運転（シミュレーション）を停止。M / ESC で閉じて運転再開
  ・画面中央に青緑色文字のリスト。↑↓で選択（押しっぱなしは3行ずつ高速閲覧）、
    Enterで再生。マウスも可: クリック=選択 / ダブルクリック=再生
  ・エンジン音など / 音楽 の音量スライダー2本を設置
  ・YouTube: IFrame APIで音声のみ再生。
    ※プレイヤーは400x300のまま画面外(top:-400px)に置く方式（ffffx.htmlと同じ）。
      1pxや透明で隠すと「隠された埋め込み」扱いでエラー150になり再生拒否されるので不可
  ・ネットラジオ: HTMLAudioで再生。HLS(.m3u8)は hls.js をCDNから自動読み込み
  ・プレイリスト行（URLなし）は今のところ選択しても何も起きない
  ・再生終了で次の曲を自動再生
  ・エラー時の挙動:
      手動選択した曲 = 飛ばさず「⚠ エラー」表示（曲名にも（エラー）付記）で停止
      連続再生中     = 通知を出して次の曲へスキップ
  ・再生中は画面最下部中央に「」の中身を青緑（21px・細字・くっきり）で表示。
    Gの表示切替（2段階目）で消える
  ・注意: ローカル(http://127.0.0.1)ではレーベル系のYouTube曲がエラー150になる
    ことがある（埋め込み自体は許可されていてもオリジン制限で弾かれる）。
    GitHub Pages（https）へ上げると再生できる可能性が高い

■ その他の画面機能
  ・V ボンネットカメラ: ボンネット先端付近から進行方向を見る一人称視点
  ・F バックミラー: 画面上部中央の枠付き小窓に後方視点（ミラー内は自車非表示）
  ・Y 自動運転: 道路中央ラインを追従、巡航130km/h、カーブ手前減速、
    直角に近い急カーブはサイドブレーキでドリフト。周回の継ぎ目は直進通過後に再同期
  ・ESC 一時停止: PAUSE表示＋操作説明を強制表示。運転キーは受け付けない

■ 運用メモ
  ・編集後は car2\car2（配布用ミラー）へ同期する
    （js/main.js, js/audio.js, index.html, README.md, musiclist.txt, vox/, sound/）
  ・index.html の main.js ?v= を変更のたびにバンプ（現在 20260718-car2-48）
  ・古いローカルサーバー（ポート8765）が残っていると変更前のファイルを配信するので、
    確認時はサーバーウィンドウを閉じて PLAY_CAR2.bat を起動し直す
  ・musiclist.txt の48行目と50行目は同じYouTube ID（木綿のハンカチーフ）に
    なっており、時の過ぎゆくままにを直すにはURLの差し替えが必要

■ 実装したが廃止した仕様（記録）
  ・壁の外を覆うモヤ（霧の層） → 廃止
  ・車の光だまり・道路照射 → ボンネットが光って見えるため廃止（現在はランプ点のみ＋自車の実光源のみ）
  ・自車のテールランプ残像（一つ分） → 廃止（残像はCPU車のみ）
  ・YouTubeの映像プレイヤー表示（右上小窓） → 音声のみに変更
