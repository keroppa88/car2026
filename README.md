# CAR2 LOOP DRIVE

`map.gltf` の高さ0にある濃いグレーの区域を、`car` の運転処理で走行する版です。
地図は元データの75%サイズで表示します（車両サイズは変更しません）。

- `PLAY_CAR2.bat` をダブルクリックして起動
- `S`: アクセル / `A`: ブレーキ
- `←` `→`: ハンドル / `↑` `↓`: シフト
- `Space`: サイドブレーキ / `T`: 全体地図
- `M`: 音楽選択(musiclist.txt の「」内を↑↓で選びEnterで再生。運転は一時停止。音量スライダー付き)
- `J`/`K`/`L`: 曲の10秒巻き戻し/再生・一時停止/10秒早送り(運転中も有効)
- `I`/`O`: 前の曲/次の曲(運転中も有効)
- `N`: 夜間モード(全車のヘッドライト・テールランプが点灯) / `B`: 音声切替(エンジン音→車内音→無音)
- `V`: カメラ切替(後方→ボンネット→視点のみ[車体非表示・マウスで視点操作]) / `G`: 表示切替5状態(操作表+音楽+スピード→音楽+スピード→スピードのみ→音楽のみ→なし)
- `F`: バックミラー / `Y`: 自動運転(平均130km/h・急カーブはドリフト) / `ESC`: 一時停止
- 南北の道路端はつながっており、一本道をそのまま走ると周回します
- ループ地点の手前から、道路の先に次の周回地図が連続して見えます
- 高さ0より上の壁・建物に当たり判定があります
- car側の緑色の補助床はcar2では表示しません
- voxフォルダ内のプレイヤー車以外をCPU車として配置し、100～150km/hのランダム速度で周回させます
- CPU車はコーナーをドリフトで曲がり、走行ラインから横±2.5mまでに制限されるため道路外へは出ません

ローカルサーバーの既定URLは `http://127.0.0.1:8765/` です。使用中の場合は次の空きポートを自動で使います。
